    - Implement IMessage and IMessageHandler

      - int routingChannelId
      - int action (0=REQUEST, 1=RELEASE, 2=RESPONSE)
      - int channelCount (for response)
    - toBytes()/fromBytes() for serialization
    - onMessage() handles:
      - REQUEST: Call controller.allocateChannels(), send RESPONSE
      - RELEASE: Call controller.releaseChannels()
      - RESPONSE: Client-side handling (update local state)

    Packet structure:
    ```java
    public class PacketRoutingChannel implements IMessage {
        private UUID terminalId;
        private int routingChannelId;
        private byte action;
        private int channelCount;
        
        public enum Action {
            REQUEST((byte)0),
            RELEASE((byte)1),
            RESPONSE((byte)2);
            
            private final byte id;
            Action(byte id) { this.id = id; }
            public byte getId() { return id; }
        }
    }
    ```

    Handler implementation:
    - Server-side: Find controller by routingChannelId, perform allocation
    - Client-side: Update local state with allocated channels
  </action>

    - Implement IGuiHandler
    - getServerGuiElement() returns ContainerRoutingController
    - getClientGuiElement() returns GuiRoutingController
    - Use GUI IDs to distinguish different GUIs

    ```java
    public class GuiHandler implements IGuiHandler {
        public static final int GUI_ROUTING_CONTROLLER = 0;
        
        @Override
        public Object getServerGuiElement(int ID, EntityPlayer player, World world, int x, int y, int z) {
            if (ID == GUI_ROUTING_CONTROLLER) {
                TileEntity te = world.getTileEntity(x, y, z);
                if (te instanceof RoutingControllerTile) {
                    return new ContainerRoutingController((RoutingControllerTile) te, player);
                }
            }
            return null;
        }
        
        @Override
        public Object getClientGuiElement(int ID, EntityPlayer player, World world, int x, int y, int z) {
            if (ID == GUI_ROUTING_CONTROLLER) {
                TileEntity te = world.getTileEntity(x, y, z);
                if (te instanceof RoutingControllerTile) {
                    return new GuiRoutingController(
                        new ContainerRoutingController((RoutingControllerTile) te, player));
                }
            }
            return null;
        }
    }
    ```

    Register in AE2ChannelRouter.java:
    ```java
    @EventHandler
    public void preInit(FMLPreInitializationEvent event) {
        // ... existing code ...
        NetworkRegistry.INSTANCE.registerGuiHandler(this, new GuiHandler());
    }
    ```
